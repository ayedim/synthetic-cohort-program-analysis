SYNTHETIC COHORT MODELING OF SEMINARY FORMATION ATTRITION

REPOSITORY OVERVIEW

This repository contains analytical materials supporting a synthetic cohort modeling project examining seminary formation attrition dynamics through rule-based data generation, institutional segmentation analysis, and dashboard reporting.

Because publicly available individual-level seminary formation microdata are generally unavailable, the project uses synthetic observations designed to simulate plausible institutional trajectories while maintaining explicit methodological constraints.

The project was developed as a descriptive institutional-analysis exercise focused on:

- synthetic cohort construction
- rule-based variable generation
- segmentation analysis
- dashboard reporting
- structured analytical documentation
- institutional-process modeling

The framework is descriptive rather than predictive and does not represent real seminarians, institutional personnel records, or empirical population estimates.

METHODOLOGICAL CONSTRAINTS

This project uses entirely synthetic data generated through rule-based logic structures informed by published aggregate reporting, seminary enrollment reporting, ordination summaries, and qualitative attrition literature.

No real seminarians, institutional personnel files, confidential records, or individual evaluations are represented.

The repository is intended solely as a methodological and analytical demonstration artifact supporting institutional modeling, descriptive analytics, segmentation reporting, and dashboard implementation.
